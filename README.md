- 🍔 Food Website (Frontend Project)

A responsive and visually appealing food ordering website UI built using HTML, CSS, and JavaScript. This project showcases a modern layout for restaurants or food delivery platforms.

🚀 Live Demo

🔗 https://sakshmaurya.github.io/demo-jms/

📌 Overview

This project is a frontend-only food website designed to simulate a restaurant or online food ordering platform. It focuses on UI/UX design, layout structure, and responsiveness.

Users can browse food items, explore menus, and interact with the interface.

✨ Features
🍕 Attractive homepage with food items
📋 Menu section showcasing dishes
🛒 Add-to-cart UI (if implemented)
📱 Responsive design (mobile-friendly)
🎨 Clean and modern layout
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & layout
JavaScript – Interactivity
